# Overlay

`Overlay` представляет собой фоновый слой, который может использоваться для создания затемнения или размытия фона. Компонент поддерживает различные варианты отображения и может быть использован для создания модальных окон, всплывающих подсказок или других элементов интерфейса, требующих визуального выделения.

- Основной сценарий использования - создание фона для всплывающих элементов (например, модальных окон или выдвижных панелей)

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=16904-214690&p=f&t=fqPE6zhGSWCbrorh-0).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Overlay } from "ics-ui-kit/components/overlay";

export const Basic = () => {
	return <Overlay />;
};
```

### Анимация

Компонент поддерживает анимацию появления и исчезновения через data-атрибуты:

- `data-state="open"` - состояние открытого оверлея, запускает анимацию появления
- `data-state="closed"` - состояние закрытого оверлея, запускает анимацию исчезновения

```tsx
import { Overlay } from "ics-ui-kit/components/overlay";

export const AnimatedOverlay = () => {
	return <Overlay data-state="open" />;
};
```

Обычно эти атрибуты управляются автоматически родительскими компонентами (например, Modal или Sheet).
При создании собственного компонента с Overlay, необходимо обеспечить правильное управление этими атрибутами:

```tsx
import { Overlay } from "ics-ui-kit/components/overlay";

export const CustomPopup = ({ isOpen, onClose }) => {
	return (
		<>
			<Overlay data-state={isOpen ? "open" : "closed"} onClick={onClose} />
			<div className="relative z-[51]">{/* Контент поверх оверлея */}</div>
		</>
	);
};
```

Важно:

- Для плавного появления/исчезновения, изменяйте `data-state` вместе с состоянием видимости компонента

### Свойства

- `type` - тип фона

    - default - стандартный
    - gradient - градиентный
    - transparent - прозрачный

- `blur` - включает эффект размытия фона

- `className` - кастомные стили

- поддерживает все стандартные свойства div-компонента, такие как `children` и другие
