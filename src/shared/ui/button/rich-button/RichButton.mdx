# RichButton

`RichButton` - это расширенная версия кнопки, которая позволяет отображать иконку, заголовок и описание в едином компоненте.

- Компонент основан на `ButtonBase`
- Полная поддержка управления с клавиатуры

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=18205-22610&m=dev)

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { RichButton } from "ics-ui-kit/components/button";
import { Flower } from "lucide-react";

export const Basic = () => {
	return <RichButton icon={Flower} title="Item title" description="Supporting text" />;
};
```
