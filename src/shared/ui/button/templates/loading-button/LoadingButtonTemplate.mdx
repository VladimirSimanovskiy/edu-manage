# LoadingButtonTemplate

`LoadingButtonTemplate` - компонент кнопки, который отображает состояние загрузки.

- Компонент основан на базовых компонентах `Button` и `IconButton`

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=34-6&p=f&m=dev)

### Базовое использование

<Canvas of={LoadingButtonTemplateStories.Default} />

```tsx
import { LoadingButtonTemplate } from "ics-ui-kit/components/button";

export const Basic = () => {
	return <LoadingButtonTemplate text="Загрузка" />;
};
```

### Варианты использования

Компонент LoadingButtonTemplate может быть использован в двух основных вариантах:

1. Как иконка загрузки без текста:

```tsx
<LoadingButtonTemplate variant="primary" status="default" size="md" />
```

2. Как кнопка с текстом и иконкой загрузки:

```tsx
<LoadingButtonTemplate variant="primary" status="default" size="md" text="Загрузка" />
```
