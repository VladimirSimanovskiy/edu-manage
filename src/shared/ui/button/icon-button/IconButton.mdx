# IconButton

`IconButton` - это специализированный компонент кнопки, предназначенный для отображения только иконки.

- Компонент основан на `ButtonBase`

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=34-6&p=f&t=LJQpzoFzCfKYGUgN-0).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { IconButton } from "ics-ui-kit/components/button";
import { Circle } from "lucide-react";

export const Basic = () => {
	return <IconButton icon={Circle} />;
};
```
