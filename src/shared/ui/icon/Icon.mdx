# Icon

`Icon` представляет собой обертку для иконок из библиотеки `Lucide`. Предоставляет базовую функциональность для отображения иконок с возможностью настройки размера.

- Использует иконки из библиотеки `Lucide`
- Основные сценарии использования:
    - Декоративные иконки
    - Иконки действий
    - Иконки статусов

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=1-433&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Base} />

```tsx
import { Icon } from "ics-ui-kit/components/icon";
import { Factory } from "lucide-react";

export const Basic = () => {
	return <Icon icon={Factory} />;
};
```

### Свойства

- `icon` - компонент иконки из библиотеки Lucide

- `size` - размер иконки

    - sm
    - md
    - lg

- `className` - кастомные стили

### Доступность

- Для улучшения доступности рекомендуется:
    - Обеспечивать достаточный контраст
    - Использовать иконки осмысленно

### Рекомендации

- Используйте иконки для улучшения визуального восприятия
- Следите за размером иконок в зависимости от контекста
- Используйте `className` для тонкой настройки
