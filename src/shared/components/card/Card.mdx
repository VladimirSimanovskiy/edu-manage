# Card

Компонент `Card` и его подкомпоненты позволяют собирать различные варианты карточек для отображения информации.

**Основные компоненты:**

- `Card` - базовый контейнер для отображения связанной информации, поддерживает состояния выбора, только для чтения и ошибки
- `ActionCard` - интерактивная кликабельная карточка с эффектами наведения и фокуса
- `CardTitle` - заголовок карточки
- `CardSubTitle` - подзаголовок карточки
- `CardFooter` - нижняя часть карточки для дополнительной информации
- `CardFeature` - область для отображения иконки или другого визуального элемента
- `CardVisualBadge` - визуальный бейдж для карточки
- `CardMenuTrigger` - триггер для меню карточки
- `CardFolder` - контейнер для создания эффекта папки

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Card } from "ics-ui-kit/components/card";

export const Basic = () => {
	return <Card>Простая карточка с текстом</Card>;
};
```
