# ScrollShadowContainer

`ScrollShadowContainer` - это компонент, который добавляет визуальные индикаторы (тени) при прокрутке контента. Компонент автоматически показывает тени сверху и снизу, когда контент можно прокрутить в соответствующем направлении.

- Основан на [Radix UI ScrollArea](https://www.radix-ui.com/primitives/docs/components/scroll-area)
- Автоматически управляет отображением теней
- Основные сценарии использования:
    - Списки с прокруткой
    - Модальные окна с большим контентом
    - Боковые панели с прокруткой
    - Любые контейнеры с ограниченной высотой

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { ScrollShadowContainer } from "ics-ui-kit/components/scroll-shadow-container";

export const Basic = () => {
	return (
		<ScrollShadowContainer className="h-[200px] w-[300px] border">
			<div className="h-[600px] w-full">{/* Длинный контент */}</div>
		</ScrollShadowContainer>
	);
};
```

### Свойства

- `wrapperClassName` - CSS классы для внешнего контейнера
- `shadowTopClassName` - CSS классы для верхней тени
- `shadowBottomClassName` - CSS классы для нижней тени
- все свойства `div` - компонента

### Доступность

- Обеспечивает достаточный контраст для видимости теней
- Правильно обрабатывает состояния прокрутки

### Рекомендации

- Используйте для контейнеров с ограниченной высотой
- Используйте в сочетании с другими компонентами, требующими прокрутки
- Учитывайте производительность при работе с большим контентом
