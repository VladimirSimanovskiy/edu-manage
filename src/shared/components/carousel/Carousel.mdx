# Carousel

`Carousel` - это компонент для создания карусели/слайдера с возможностью горизонтальной и вертикальной прокрутки.

- Компонент основан на [Carousel из shadcn](https://ui.shadcn.com/docs/components/carousel)
- Компонент использует библиотеку [Embla Carousel](https://www.embla-carousel.com/)
- Поддерживает навигацию с помощью кнопок и клавиатуры
- Поддерживает прокрутку колесом мыши и жестами тачпада (можно отключить через `disableWheelScroll`)

## Примеры

### Горизонтальная карусель с фиксированной шириной

<Canvas of={CarouselStories.Horizontal} />

```tsx
import {
	Carousel,
	CarouselContent,
	CarouselItem,
	CarouselPrevious,
	CarouselNext
} from "ics-ui-kit/components/carousel";
import { Card } from "ics-ui-kit/components/card";

const items = Array.from({ length: 10 }, (_, i) => ({
	title: `${i + 1}`
}));

export const FixedWidthCarousel = () => {
	return (
		<div className="w-[500px]">
			<Carousel orientation="horizontal" size="md">
				<CarouselContent>
					{items.map((item, index) => (
						<CarouselItem key={index}>
							<Card className="pointer-events-none flex h-full items-center justify-center rounded-xl p-4 shadow-none">
								<h3 className="text-lg font-semibold">{item.title}</h3>
							</Card>
						</CarouselItem>
					))}
				</CarouselContent>
				<CarouselPrevious />
				<CarouselNext />
			</Carousel>
		</div>
	);
};
```
