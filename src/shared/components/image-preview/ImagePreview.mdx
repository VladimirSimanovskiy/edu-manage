# ImagePreview

`ImagePreview` - это компонент для отображения предпросмотра изображения с возможностью скачивания, удаления и повторной загрузки.

- Имеется поддержка управления с клавиатуры с визуальной индикацией фокуса

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=20678-33021&p=f&t=AhH3bQrmUSZherio-0).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { ImagePreview } from "ics-ui-kit/components/image-preview";

function MyComponent() {
	return (
		<ImagePreview
			url="https://placehold.co/200x200/png"
			onClear={() => alert("Вызов метода очищения")}
			onRetry={() => alert("Повторная попытка загрузки")}
			onDownload={() => alert("Скачивание изображения")}
			onImageClick={() => alert("Открытие изображения")}
			size="sm"
		/>
	);
}
```
