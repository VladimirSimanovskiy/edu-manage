# Scheduler

`Scheduler` представляет собой планировщик событий с возможностью перетаскивания и изменения размера событий. Компонент поддерживает отображение событий на временной шкале с возможностью настройки интервалов времени и высоты строк.

- Поддерживает отображение событий на временной шкале
- Позволяет перетаскивать и изменять размер событий
- Настраиваемые интервалы времени и высота строк
- Поддерживает отображение одного или нескольких дней
- Интегрируется с `Tooltip` для отображения дополнительной информации
- Полностью доступен для клавиатурной навигации

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Scheduler } from "ics-ui-kit/components/scheduler";

export const Basic = () => {
	return (
		<Scheduler
			startDate={new Date()}
			endDate={new Date(Date.now() + 7 * 24 * 60 * 60 * 1000)}
			events={[
				{
					id: "1",
					title: "Событие",
					startDate: new Date(),
					endDate: new Date(Date.now() + 60 * 60 * 1000)
				}
			]}
		/>
	);
};
```

### Свойства

#### Scheduler

- `events` - список отображаемых событий
- `startDate` - дата с которой начинается отображение
- `endDate` - дата с которой заканчивается отображение
- `timeSnap` - интервал округления времени в минутах
- `hourHeight` - высота строки часа в px
- `showDates` - показывать дни в заголовке
- `onDayClick` - событие при клике на день
- `onEventClick` - событие при клике на событие
- `onEventDrop` - событие при перетаскивании события
- `onEventResize` - событие при изменении размера события
- `eventComponent` - компонент для отображения события
- `isDraggable` - функция, определяющая возможность перетаскивания
- `isResizable` - функция, определяющая возможность изменения размера

### Доступность

- Обеспечивает правильную фокусировку и навигацию
