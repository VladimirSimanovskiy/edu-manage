# EmptyState

`EmptyState` представляет собой компонент для отображения пустого состояния интерфейса, когда нет данных для показа. Компонент обеспечивает единообразное отображение пустых состояний во всем приложении.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=118-2682&p=f&t=fqPE6zhGSWCbrorh-0).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { EmptyState } from "ics-ui-kit/components/empty-state";

export const Basic = () => {
	return <EmptyState>No results found</EmptyState>;
};
```
