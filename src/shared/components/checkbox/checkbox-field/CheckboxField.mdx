# CheckboxField

`CheckboxField` расширяет базовый `Checkbox`, добавляя метку, описание и дополнительные стили. Предоставляет более удобный способ создания форм с чекбоксами, включая поддержку состояний ошибки и отключения.

## Использование

- Компонент основан на базовом `Checkbox`, использует [Radix UI Checkbox](https://www.radix-ui.com/primitives/docs/components/checkbox)
- Основные сценарии использования:
    - Формы с чекбоксами
    - Соглашения и условия
    - Группы связанных чекбоксов

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=46-67&p=f&t=fqPE6zhGSWCbrorh-0).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { CheckboxField } from "ics-ui-kit/components/checkbox";

export const Basic = () => {
	return (
		<CheckboxField
			label="Accept terms and conditions"
			description="By accepting, you agree to our privacy policy and terms of service."
		/>
	);
};
```

### Свойства

- `label` - текст метки чекбокса

- `description` - описание чекбокса

- `alignment` - выравнивание чекбокса относительно метки

    - right
    - left

- `outline` - добавление контура вокруг чекбокса

- `error` - состояние ошибки

- `disabled` - отключение возможности переключения

- поддерживает все стандартные свойства `CheckboxPrimitive.Root`

### Доступность

- Компонент наследует все возможности доступности от `Checkbox`
- Автоматически связывает метку с чекбоксом
- Для улучшения доступности рекомендуется:
    - Использовать описательные метки
    - Добавлять описания для сложных чекбоксов
    - Обеспечивать достаточный контраст для состояний

### Рекомендации

- Используйте лейблы для всех чекбоксов
- Выбирайте подходящее выравнивание в зависимости от контекста
- Используйте состояние ошибки для валидации
- Используйте `className` для тонкой настройки стилей
