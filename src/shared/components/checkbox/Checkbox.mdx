# Checkbox

`Checkbox` представляет собой базовый чекбокс, основанный на [Radix UI Checkbox](https://www.radix-ui.com/primitives/docs/components/checkbox). Поддерживает три состояния: выбранный, невыбранный и неопределенный (indeterminate). Компонент полностью доступен и поддерживает клавиатурную навигацию.

- Компонент основан на `Radix UI Checkbox`
- Основные сценарии использования:
    - Одиночный выбор
    - Группы чекбоксов

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=46-67&p=f&t=fqPE6zhGSWCbrorh-0).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Checkbox } from "ics-ui-kit/components/checkbox";

export const Basic = () => {
	return <Checkbox />;
};
```

### Свойства

- `checked` - состояние чекбокса
    - boolean
    - "indeterminate"
- поддерживает все стандартные свойства `CheckboxPrimitive.Root`

### Доступность

- Компонент полностью доступен и поддерживает:
    - Клавиатурную навигацию (Tab, Space, Enter)
    - Программы чтения с экрана
    - Состояния фокуса и ховера

### Рекомендации

- Используйте `className` для тонкой настройки стилей
