# InputOTP

`InputOTP` - это компонент для ввода одноразовых паролей (OTP), который позволяет пользователям вводить код подтверждения по символам.

- Компонент сделан на основе [shadcn/ui input-otp](https://ui.shadcn.com/docs/components/input-otp)
- Компонент использует библиотеку [input-otp](https://github.com/guilhermerodz/input-otp)
- Поддерживает автофокус, клавиатурную навигацию и вставку кода из буфера обмена

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=21408-315999&m=dev)

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { InputOTP, InputOTPGroup, InputOTPSlot } from "ics-ui-kit/components/input-otp";

export const Basic = () => {
	return (
		<InputOTP maxLength={6}>
			<InputOTPGroup>
				<InputOTPSlot index={0} />
				<InputOTPSlot index={1} />
				<InputOTPSlot index={2} />
			</InputOTPGroup>
			<InputOTPSeparator />
			<InputOTPGroup>
				<InputOTPSlot index={3} />
				<InputOTPSlot index={4} />
				<InputOTPSlot index={5} />
			</InputOTPGroup>
		</InputOTP>
	);
};
```

### Поддержка спецсимволов при вставке

Компонент поддерживает автоматическую обработку специальных символов при вставке текста. Вы можете настроить, какие символы нужно удалять при вставке, используя проп `pasteTransformer`.

Примеры использования:

```tsx
// Удаление всех нецифровых символов
<InputOTP pasteTransformer={(pasted) => pasted.replace(/\D/g, '')} />

// Удаление только точек и дефисов
<InputOTP pasteTransformer={(pasted) => pasted.replace(/[.-]/g, '')} />
```
