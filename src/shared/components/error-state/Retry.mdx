# Retry

`Retry` — это специализированный компонент кнопки для повторного выполнения действий после ошибки или неудачной операции.

Компонент предназначен для предоставления пользователю возможности повторить неуспешную операцию.

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Retry } from "ics-ui-kit/components/error-state";
import { RotateCcw } from "lucide-react";

export const Basic = () => {
	const handleRetry = () => {
		// Логика повторного выполнения
		console.log("Retrying...");
	};

	return <Retry onClick={handleRetry}>Try again</Retry>;
};
```
