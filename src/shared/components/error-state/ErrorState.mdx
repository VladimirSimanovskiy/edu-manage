# ErrorState

`ErrorState` представляет собой компонент для отображения состояния ошибки в интерфейсе, когда что-то пошло не так.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=118-2682&p=f&t=fqPE6zhGSWCbrorh-0).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { ErrorState } from "ics-ui-kit/components/error-state";

export const Basic = () => {
	return <ErrorState>Something went wrong</ErrorState>;
};
```
