# Sidebar

`Sidebar` - это гибкий компонент боковой панели, который обеспечивает навигацию и организацию контента в приложении.

- Поддерживает автоматическое сворачивание на мобильных устройствах
- Поддержка клавиатурных сокращений (Cmd/Ctrl + B)
- Различные варианты отображения: sidebar, floating, inset
- Режимы сворачивания: offcanvas, icon, none

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/01_UI-kit?node-id=5198-1148&m=dev).

## Структура компонента

Sidebar состоит из следующих подкомпонентов:

- `SidebarProvider` - провайдер контекста для управления состоянием
- `Sidebar` - основной компонент боковой панели
- `SidebarHeader` - заголовок боковой панели
- `SidebarContent` - основное содержимое
- `SidebarFooter` - подвал боковой панели
- `SidebarGroup` - группа элементов меню
- `SidebarGroupLabel` - заголовок группы
- `SidebarGroupContent` - содержимое группы
- `SidebarGroupAction` - действие для группы
- `SidebarMenu` - контейнер для пунктов меню
- `SidebarMenuItem` - элемент меню
- `SidebarMenuButton` - кнопка пункта меню
- `SidebarMenuAction` - действие для пункта меню
- `SidebarMenuBadge` - бейдж для пункта меню
- `SidebarMenuSub` - подменю
- `SidebarMenuSubItem` - элемент подменю
- `SidebarMenuSubButton` - кнопка элемента подменю
- `SidebarMenuSkeleton` - скелетон для загрузки
- `SidebarSeparator` - разделитель
- `SidebarInput` - поле ввода
- `SidebarTrigger` - триггер для открытия/закрытия
- `SidebarRail` - рельс для изменения размера
- `SidebarInset` - основная область контента

### Базовое использование

<Canvas of={SidebarStories.Default} />

```tsx
import {
	Sidebar,
	SidebarProvider,
	SidebarHeader,
	SidebarContent,
	SidebarGroup,
	SidebarGroupLabel,
	SidebarGroupContent,
	SidebarMenu,
	SidebarMenuItem,
	SidebarMenuButton,
	SidebarSeparator
} from "ics-ui-kit/components/sidebar";
import { Home, Inbox } from "lucide-react";

export const Basic = () => {
	return (
		<SidebarProvider>
			<Sidebar>
				<SidebarHeader>Мое приложение</SidebarHeader>
				<SidebarSeparator />
				<SidebarContent>
					<SidebarGroup>
						<SidebarGroupLabel>Навигация</SidebarGroupLabel>
						<SidebarGroupContent>
							<SidebarMenu>
								<SidebarMenuItem>
									<SidebarMenuButton asChild>
										<a href="#">
											<Home />
											<span>Главная</span>
										</a>
									</SidebarMenuButton>
								</SidebarMenuItem>
								<SidebarMenuItem>
									<SidebarMenuButton asChild>
										<a href="#">
											<Inbox />
											<span>Входящие</span>
										</a>
									</SidebarMenuButton>
								</SidebarMenuItem>
							</SidebarMenu>
						</SidebarGroupContent>
					</SidebarGroup>
				</SidebarContent>
			</Sidebar>
		</SidebarProvider>
	);
};
```
