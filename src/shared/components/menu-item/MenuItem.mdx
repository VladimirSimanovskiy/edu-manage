# MenuItem

`MenuItem` - система компонентов для создания элементов меню с различными типами контента и взаимодействий.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=16852-2888&p=f&m=dev).

## Основные компоненты

### MenuItem

Базовый контейнер для элементов меню с поддержкой hover эффектов и состояний disabled.

### MenuItemAction

Интерактивный элемент меню с поддержкой фокуса и клавиатурной навигации.

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { MenuItem, MenuItemIcon, MenuItemText } from "ics-ui-kit/components/menu-item";
import { Circle } from "lucide-react";

export const Basic = () => {
	return (
		<MenuItem>
			<MenuItemIcon icon={Circle} />
			<MenuItemText>Menu text</MenuItemText>
		</MenuItem>
	);
};
```

## Вспомогательные компоненты

- **MenuItemIcon** - контейнер для иконок
- **MenuItemText** - основной текст элемента
- **MenuItemSecondaryText** - дополнительный текст
- **MenuItemShortcut** - горячие клавиши
- **MenuItemButton** - кнопка действия
- **MenuItemIconButton** - кнопка с иконкой
