# Label

`Label` представляет собой базовый лейбл, обеспечивающий доступность и правильную связь с элементами формы через атрибут `htmlFor`.

- Основан на [Radix UI Label](https://www.radix-ui.com/primitives/docs/components/label)
- Правильно связывается с элементами формы через `htmlFor`

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=65-517&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Base} />

```tsx
import { Label } from "ics-ui-kit/components/label";

export const Basic = () => {
	return <Label>Username</Label>;
};
```
