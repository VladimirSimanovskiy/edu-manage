# FieldLabel

`FieldLabel` представляет собой расширенную версию базового `Label`, специально разработанную для использования с полями формы.

- Основан на компоненте `Label`
- Автоматически отображает астериск для обязательных полей
- Поддерживает иконки с опциональным тултипом

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=65-517&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { FieldLabel } from "ics-ui-kit/components/label";

export const Basic = () => {
	return <FieldLabel>Лейбл поля</FieldLabel>;
};
```

### Свойства

| Свойство   | Тип          | Описание                                              |
| ---------- | ------------ | ----------------------------------------------------- |
| `required` | `boolean`    | Флаг, указывающий, что поле является обязательным     |
| `icon`     | `LucideIcon` | Иконка из библиотеки Lucide Icons                     |
| `tooltip`  | `string`     | Текст подсказки, отображаемой при наведении на иконку |

Компонент также поддерживает все стандартные атрибуты `Label`.
