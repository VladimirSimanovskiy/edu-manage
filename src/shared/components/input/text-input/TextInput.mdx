# TextInput

`TextInput` представляет собой расширенную версию базового `Input` с поддержкой иконок и дополнительных действий.

- Основан на базовом компоненте `Input`
- Кнопки действий доступны с клавиатур

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=65-520&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { TextInput } from "ics-ui-kit/components/input";

export const Basic = () => {
	return <TextInput placeholder="Enter your text here" />;
};
```
