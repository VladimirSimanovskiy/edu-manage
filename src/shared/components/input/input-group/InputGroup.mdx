# InputGroup

Компонент для создания групп инпутов с дополнительными элементами (текст, кнопки, иконки).
Позволяет легко добавлять префиксы, суффиксы и кнопки к полям ввода.

## Использование

```tsx
import { InputGroup, InputGroupText, InputGroupInput, InputGroupButton } from 'ics-ui-kit/components/input';

// Базовое использование с префиксом
<InputGroup>
  <InputGroupText>@</InputGroupText>
  <InputGroupInput placeholder="Username" />
</InputGroup>

// С суффиксом
<InputGroup>
  <InputGroupInput placeholder="Amount" />
  <InputGroupText>.00</InputGroupText>
</InputGroup>

// С кнопкой
<InputGroup>
  <InputGroupInput placeholder="Search..." />
  <InputGroupButton>Search</InputGroupButton>
</InputGroup>
```
