# Input

`Input` представляет собой стилизованное текстовое поле ввода. Компонент обеспечивает базовую функциональность для ввода текста с поддержкой различных состояний (disabled, readOnly, invalid) и настраиваемыми стилями.

- Компонент основан на HTML `input` элементе
- Поддерживает все стандартные HTML атрибуты input
- Основные сценарии использования:
    - Базовые поля ввода
    - Поля с валидацией
    - Поля только для чтения
    - Отключенные поля

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=65-520&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Input } from "ics-ui-kit/components/input";

export const Basic = () => {
	return <Input placeholder="Placeholder" />;
};
```

### Только для чтения

<Canvas of={ComponentStories.ReadOnly} />

```tsx
import { Input } from "ics-ui-kit/components/input";

export const ReadOnly = () => {
	return <Input defaultValue="Some text" readOnly />;
};
```

### Отключенное состояние

<Canvas of={ComponentStories.Disabled} />

```tsx
import { Input } from "ics-ui-kit/components/input";

export const Disabled = () => {
	return <Input defaultValue="Some text" disabled />;

```

### Ошибка валидации

```tsx
import { Input } from "ics-ui-kit/components/input";

export const Invalid = () => {
	return <Input placeholder="Placeholder" aria-invalid={true} aria-errormessage="Error message" />;
};
```

<Canvas of={ComponentStories.NativeValidation} />

```tsx

	return <Input placeholder="Placeholder" minLength={3} />;

```

### Кастомные стили

		<Input

### Свойства

- компонент принимает все стандартные HTML атрибуты `input` элемента

### Доступность

- Имеет встроенную поддержку клавиатурной навигации

### Рекомендации

- Используйте соответствующие типы `input` для разных видов данных
- Используйте валидацию для обеспечения корректности данных
- Применяйте кастомные стили через `className` для тонкой настройки
- Обеспечьте достаточный контраст для видимости
