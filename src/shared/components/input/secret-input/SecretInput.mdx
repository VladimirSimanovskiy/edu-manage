# SecretInput

`SecretInput` представляет собой специализированную версию `TextInput` для ввода конфиденциальных данных, таких как пароли и токены. Компонент обеспечивает базовую функциональность скрытия/показа введенного текста, а также включает дополнительные функции безопасности.

- Основан на компоненте `TextInput`
- Поддерживает иконки из библиотеки `Lucide`
- Основные сценарии использования:
    - Поля ввода паролей
    - Поля ввода токенов
    - Поля ввода конфиденциальных данных
    - Поля с повышенными требованиями к безопасности

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=65-520&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { SecretInput } from "ics-ui-kit/components/input";

export const Basic = () => {
	return <SecretInput placeholder="Enter your password" />;
};
```
