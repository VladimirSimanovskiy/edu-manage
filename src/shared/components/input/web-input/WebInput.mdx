# WebInput

`WebInput` представляет собой специализированный компонент для ввода веб-адресов. Основан на `InputGroup` компонентах и включает функционал префикса, очистки и копирования значения.

- Поддерживает настраиваемый префикс (по умолчанию "https://")
- Включает кнопки очистки и копирования
- Работает как в controlled, так и в uncontrolled режиме

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { WebInput } from "ics-ui-kit/components/input";

export const Basic = () => {
	return <WebInput placeholder="example.com" />;
};
```

### Свойства

| Свойство       | Описание                                |
| -------------- | --------------------------------------- |
| `startContent` | Содержимое в начале поля                |
| `showClearIcon`| Флаг отображения кнопки очистки         |
| `showCopyIcon` | Флаг отображения кнопки копирования     |
| `onChange`     | Обработчик изменения значения           |
| `error`        | Текст сообщения об ошибке               |
| ...            | Все стандартные атрибуты input элемента |
