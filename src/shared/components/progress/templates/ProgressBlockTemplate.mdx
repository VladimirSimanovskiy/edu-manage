# ProgressBlockTemplate

`ProgressBlockTemplate` - готовый к использованию шаблон блочного индикатора прогресса с заголовком, описанием и иконкой.

- Компонент основан на базовых компонентах `Progress`, `ProgressIcon`, `ProgressText` и `ProgressIconButton`

### Базовое использование

<Canvas of={ProgressBlockTemplateStories.Default} />

```tsx
import { ProgressBlockTemplate } from "ics-ui-kit/components/progress";
import { Download } from "lucide-react";

export const Basic = () => {
	return <ProgressBlockTemplate value={50} icon={Download} title="Загрузка файла" description="50%" size="md" />;
};
```
