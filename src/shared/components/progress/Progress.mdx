# Progress

`Progress` - компонент для отображения индикатора выполнения операции.

- Компонент основан на [Radix UI Progress](https://www.radix-ui.com/primitives/docs/components/progress)
- Поддерживает controlled режим
- Полная поддержка ARIA атрибутов и доступности
- Автоматическое вычисление позиции индикатора на основе значения value
- Поддержка клавиатуры: полная поддержка доступности
- Поддержка ARIA атрибутов: `aria-valuenow`, `aria-valuemin`, `aria-valuemax`

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=24273-76456&m=dev)

### Базовое использование

<Canvas of={ProgressStories.Default} />

```tsx
import { Progress } from "ics-ui-kit/components/progress";

export const Basic = () => {
	const [value, setValue] = React.useState(0);

	React.useEffect(() => {
		const interval = setInterval(() => {
			setValue((v) => (v >= 100 ? 0 : v + 10));
		}, 500);

		return () => clearInterval(interval);
	}, []);

	return <Progress value={value} className="w-[60%]" />;
};
```
