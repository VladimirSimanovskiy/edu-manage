# ProgressIconButton

`ProgressIconButton` - интерактивная кнопка-иконка, предназначенная для использования в контексте прогресса.

- Расширяет базовый компонент `IconButton`
- Поддержка клавиатуры: полная навигация и активация через Tab и Enter/Space

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=24273-76456&m=dev)

### Базовое использование

<Canvas of={ProgressIconButtonStories.Default} />

```tsx
import { ProgressIconButton } from "ics-ui-kit/components/progress";
import { X } from "lucide-react";

export const Basic = () => {
	return <ProgressIconButton icon={X} size="md" onClick={() => console.log("Click!")} />;
};
```
