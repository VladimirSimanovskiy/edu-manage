# ProgressIcon

`ProgressIcon` - компонент для отображения иконок в контексте прогресса.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=24273-76456&m=dev)

- Наследует все возможности компонента `Icon`

### Базовое использование

<Canvas of={ProgressIconStories.Default} />

```tsx
import { ProgressIcon } from "ics-ui-kit/components/progress";
import { CircleAlert } from "lucide-react";

export const Basic = () => {
	return <ProgressIcon icon={CircleAlert} size="md" />;
};
```
