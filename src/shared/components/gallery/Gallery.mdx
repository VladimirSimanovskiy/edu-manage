# Gallery

`Gallery` - компонент галереи для отображения и управления коллекцией изображений. 

- Поддерживает загрузку, предпросмотр, удаление, скачивание изображений.
- Построена на основе компонента `Carousel`
- Имеет встроенный предпросмотр изображений в модальном окне
- Поддержка управления с клавиатуры
- Поддержка `controlled` состояния

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=19733-57284&m=dev)

### Базовое использование

<Canvas of={GalleryStories.Default} />

```tsx
import { Gallery } from "ics-ui-kit/components/gallery/Gallery";

export const Basic = () => {
    const [images, setImages] = useState<Array<{ id: number; url: string; name: string }>>([]);
    return <Gallery size="sm" values={images} onValueChange={setImages} />;
};
```
