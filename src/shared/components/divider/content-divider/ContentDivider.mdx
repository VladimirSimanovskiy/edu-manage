# ContentDivider

`ContentDivider` - компонент используется для разделения контента в интерфейсе. Может содержать текст или другой контент, который объясняет причину разделения.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=118-2682&p=f&t=fqPE6zhGSWCbrorh-0).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { ContentDivider } from "ics-ui-kit/components/divider";
import { Description } from "ics-ui-kit/components/description";

export const Basic = () => {
	return (
		<ContentDivider>
			<Description>или</Description>
		</ContentDivider>
	);
};
```
