# OptionGroup

`OptionGroup` - это компонент для создания группы переключателей.

- Основан на компоненте `ToggleGroup`
- Переключатели могут быть горизонтальными или вертикальными

### Базовое использование

<Canvas of={OptionGroupStories.Default} />

```tsx
import { OptionGroup, OptionGroupItem } from "ics-ui-kit/components/option-group";

const YesNoSelector = () => {
	return (
		<OptionGroup type="single">
			<OptionGroupItem value="true">Да</OptionGroupItem>
			<OptionGroupItem value="false">Нет</OptionGroupItem>
			<OptionGroupItem value="indeterminate">Наверное</OptionGroupItem>
		</OptionGroup>
	);
};
```

### Свойства

#### OptionGroup

| Свойство    | Тип                          | Описание                                           |
| ----------- | ---------------------------- | -------------------------------------------------- |
| orientation | `"horizontal" \| "vertical"` | Ориентация группы: горизонтальная или вертикальная |
| ...props    | `ToggleGroupPrimitive.Root`  | Все свойства, имеющиеся у компонента ToggleGroup   |
