# ImagePlaceholder

`ImagePlaceholder` - это компонент для отображения пустого места под фотографию или изображение.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=20797-21811&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { ImagePlaceholder } from "ics-ui-kit/components/image-placeholder";

function MyComponent() {
	return <ImagePlaceholder size="sm" />;
}
```
