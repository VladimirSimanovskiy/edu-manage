# Prose

`Prose` — это компонент-обёртка для применения типографических стилей к HTML-контенту. Использует плагин [@tailwindcss/typography](https://github.com/tailwindlabs/tailwindcss-typography) для красивого форматирования текста.

Компонент автоматически применяет стили к заголовкам, параграфам, спискам, ссылкам, цитатам, таблицам, блокам кода и другим HTML-элементам.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=23-995&p=f&t=fqPE6zhGSWCbrorh-0).

### Базовое использование

<Canvas of={ComponentStories.TypographyShowcase} />

```tsx
import { Prose } from "ics-ui-kit/components/prose";

export const Basic = () => {
	return (
		<Prose>
			<h1>Заголовок</h1>
			<p>
				Параграф с <strong>жирным</strong> и <em>курсивным</em> текстом.
			</p>
			<ul>
				<li>Элемент списка 1</li>
				<li>Элемент списка 2</li>
			</ul>
		</Prose>
	);
};
```

## Особенности

- **Автоматическая типографика**: Применяет согласованные стили ко всем HTML-элементам
- **Адаптивность**: Размеры и отступы автоматически адаптируются под размер экрана
- **Плагины Tailwind**: Использует официальный плагин @tailwindcss/typography
- **Slot-совместимость**: Поддерживает использование с Radix UI Slot для композиции

## Поддерживаемые элементы

Компонент автоматически стилизует следующие HTML-элементы:

- **Заголовки**: `h1`, `h2`, `h3`, `h4`, `h5`, `h6`
- **Текст**: `p`, `strong`, `em`, `code`, `mark`, `del`, `ins`
- **Ссылки**: `a`
- **Списки**: `ul`, `ol`, `li`
- **Цитаты**: `blockquote`
- **Код**: `pre`, `code`, `kbd`
- **Таблицы**: `table`, `thead`, `tbody`, `tr`, `th`, `td`
- **Изображения**: `img`, `figure`, `figcaption`
- **Разделители**: `hr`
- **Дополнительно**: `abbr`, `sub`, `sup`, `small`

## Рекомендации

- Используйте для контента из CMS, Markdown или любого HTML, который вы не контролируете
- Компонент идеален для статей, документации, блогов и длинных текстов
- Для лучшей читаемости ограничивайте ширину контейнера с помощью `max-w-prose` или подобных классов
- Можно комбинировать с другими Tailwind классами для дополнительной настройки
