# PageState

`PageState` — это универсальный компонент для отображения состояний страницы или раздела приложения.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/01_UI-kit?node-id=24871-441181&m=dev).

### Базовое использование

<Canvas of={PageStateStories.Default} />

```tsx
import {
	PageState,
	PageStateTitle,
	PageStateDescription,
	PageStateIllustrationBoxSvg,
	PageStateFeatureIcon,
	PageStateButtonGroup,
	PageStateCancel,
	PageStateAction
} from "ics-ui-kit/components/page-state";
import { UploadCloud } from "lucide-react";

export const Basic = () => {
	return (
		<PageState>
			<PageStateIllustrationBoxSvg icon={<PageStateFeatureIcon icon={UploadCloud} />} />
			<PageStateTitle>Page State Title</PageStateTitle>
			<PageStateDescription>
				Add a brief, empathetic explanation that acknowledges the error while maintaining brand voice. Keep it
				light and helpful
			</PageStateDescription>
			<PageStateButtonGroup>
				<PageStateCancel>Secondary action</PageStateCancel>
				<PageStateAction>Primary action</PageStateAction>
			</PageStateButtonGroup>
		</PageState>
	);
};
```

### Компоненты системы

`PageState` состоит из набора взаимосвязанных компонентов:

#### Основной контейнер

- **`PageState`** - основной контейнер компонента

#### Иллюстрации

- **`PageStateBoxSvg`** - иллюстрация коробки
- **`PageStateFolderSvg`** - иллюстрация папки

#### Контент

- **`PageStateTitle`** - заголовок состояния
- **`PageStateDescription`** - описание состояния
- **`PageStateFeatureIcon`** - иконка особенности (отображается в центре иллюстрации)

#### Действия

- **`PageStateButtonGroup`** - группа кнопок действий
- **`PageStateAction`** - основное действие (primary button)
- **`PageStateCancel`** - вторичное действие (secondary button)
