# FileUpload

`FileUpload` - компонент для загрузки файлов с поддержкой drag & drop, валидации размера и типов файлов.

- Поддержка drag & drop и клика для выбора файлов
- Валидация максимального размера файлов и их количества
- Фильтрация по типу файлов (accept)
- Controlled и uncontrolled режимы работы
- Визуальные состояния перетаскивания и ошибок
- Доступность (ARIA-поддержка)
