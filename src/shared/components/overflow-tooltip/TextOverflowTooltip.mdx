# TextOverflowTooltip

`TextOverflowTooltip` - это компонент для отображения текста с автоматическим показом тултипа при переполнении контейнера. 

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { TextOverflowTooltip } from "ics-ui-kit/components/overflow-tooltip";

export const Basic = () => {
	return (
		<div className="w-48 rounded border border-gray-200 p-4">
			<TextOverflowTooltip>
				Это очень длинный текст, который не поместится в ограниченное пространство и будет обрезан
			</TextOverflowTooltip>
		</div>
	);
};
```
