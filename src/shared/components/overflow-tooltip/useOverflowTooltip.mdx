# useOverflowTooltip

Хук для создания тултипа, который показывается только при переполнении контента.

## Назначение

`useOverflowTooltip` определяет, переполняет ли элемент свои границы, и управляет состоянием тултипа соответственно. Тултип показывается только тогда, когда контент действительно не помещается.

## Пример использования

```tsx
import { useOverflowTooltip } from "./useOverflowTooltip";

const MyComponent = () => {
	const { open, onOpenChange, ref } = useOverflowTooltip<HTMLDivElement>();

	return (
		<Tooltip open={open} onOpenChange={onOpenChange}>
			<TooltipTrigger asChild>
				<div ref={ref}>Длинный текст, который не помещается в контейнер</div>
			</TooltipTrigger>
			<TooltipContent>
				Полный текст, который отображается в тултипе только когда элемент не помещается
			</TooltipContent>
		</Tooltip>
	);
};
```

## Как работает

1. При попытке открыть тултип проверяет переполнение элемента
2. Если элемент переполняет границы - показывает тултип
3. Если не переполняет - тултип остается скрытым
