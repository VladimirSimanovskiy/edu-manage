# SwitchField

`SwitchField` - это составной компонент, объединяющий `Switch` с меткой и описанием. Он предоставляет удобный интерфейс для создания переключателей с дополнительной информацией и поддержкой различных состояний.

## Использование

- Основан на компоненте `Switch`
- Поддерживает метку и описание
- Основные сценарии использования:
    - Настройки с пояснениями
    - Формы с переключателями
    - Соглашения и условия
    - Управление функциями с описанием

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=60-438&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { SwitchField } from "ics-ui-kit/components/switch/switch-field";

export const Basic = () => {
	return (
		<SwitchField
			label="Accept terms and conditions"
			description="By accepting, you agree to our privacy policy and terms of service."
		/>
	);
};
```

### Свойства

- `label` - текст метки чекбокса

- `description` - описание чекбокса

- `alignment` - выравнивание чекбокса относительно метки

    - right
    - left

- `outline` - добавление контура вокруг чекбокса

- `error` - состояние ошибки

- `disabled` - отключение возможности переключения

- поддерживает все стандартные свойства `SwitchPrimitives.Root`

### Доступность

- Правильно связывает метку с переключателем
- Поддерживает навигацию с клавиатуры
- Обеспечивает достаточный контраст
- Правильно обрабатывает состояния отключения и ошибок
- Поддерживает скринридеры

### Рекомендации

- Используйте для переключателей с дополнительной информацией
- Используйте состояние ошибки для невалидных значений
- Выбирайте подходящее выравнивание в зависимости от контекста
- Используйте обводку для выделения важных опций
- Используйте в сочетании с другими компонентами форм
- Используйте `className` для тонкой настройки стилей
