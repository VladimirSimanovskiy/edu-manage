# Switch

`Switch` - это компонент переключателя, основанный на [Radix UI Switch](https://www.radix-ui.com/primitives/docs/components/switch). Он предоставляет удобный интерфейс для переключения между двумя состояниями (включено/выключено). Компонент поддерживает различные размеры, состояния отключения и кастомные стили.

- Основан на `Radix UI Switch`
- Основные сценарии использования:
    - Переключение режимов
    - Управление настройками
    - Формы с булевыми значениями

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=60-438&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Switch } from "ics-ui-kit/components/switch";

export const Basic = () => {
	return <Switch />;
};
```

### Свойства

- `size` - размер переключателя

    - sm
    - md

- Все свойства `SwitchPrimitives.Root`

### Доступность

- Правильно управляет фокусом
- Поддерживает навигацию с клавиатуры
- Правильно обрабатывает состояния отключения
- Поддерживает скринридеры

### Рекомендации

- Используйте для переключения между двумя состояниями
- Выбирайте подходящий размер в зависимости от контекста
- Используйте состояние отключения для недоступных опций
- Используйте в сочетании с другими компонентами форм
- Добавляйте кастомные стили при необходимости
