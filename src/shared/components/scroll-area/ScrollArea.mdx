# ScrollArea

`ScrollArea` представляет собой кастомную область прокрутки с плавной анимацией и кастомными стилями. Компонент основан на [Radix UI ScrollArea](https://www.radix-ui.com/primitives/docs/components/scroll-area) и предоставляет вертикальную и горизонтальную прокрутку с кастомными полосами прокрутки.

- Компонент основан на `Radix UI ScrollArea`
- Предоставляет кастомные стили для полос прокрутки
- Поддерживает как вертикальную, так и горизонтальную прокрутку
- Имеет плавную анимацию появления/исчезновения полос прокрутки

**⚠️ ВАЖНО!**
Этот компонент предназначен только для внутреннего использования в рамках библиотеки. Он не экспортируется наружу и не должен использоваться в других проектах.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=296-207&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { ScrollArea } from "@/components/scroll-area/ScrollArea";

export const Basic = () => {
	return (
		<ScrollArea className="h-[200px] w-[300px] border">
			<div className="h-[600px] w-[600px]">{/* Контент, который требует прокрутки */}</div>
		</ScrollArea>
	);
};
```

### Свойства

#### ScrollArea

- использует все свойства `ScrollAreaPrimitive.Root`

#### ScrollBar

- использует все свойства `ScrollAreaPrimitive.ScrollAreaScrollbar`

### Доступность

- Автоматически управляет состоянием полос прокрутки

### Рекомендации

- Используйте для контента, который может не помещаться в видимую область
