# Collapsible

`Collapsible` представляет собой компонент для создания раскрывающихся секций контента. Позволяет пользователям показывать и скрывать дополнительную информацию по клику на триггер.

- Компонент основан на [shadcn/ui](https://ui.shadcn.com/docs/components/collapsible)
- Компонент использует [@radix-ui/react-collapsible](https://www.radix-ui.com/primitives/docs/components/collapsible)
- Компонент автоматически обрабатывает ARIA атрибуты и предоставляет корректные роли для доступности
- Поддерживает навигацию с клавиатуры

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Collapsible, CollapsibleTrigger, CollapsibleContent } from "ics-ui-kit/components/collapsible";

export const Example = () => {
	return (
		<Collapsible>
			<CollapsibleTrigger>Что такое компонент Collapsible?</CollapsibleTrigger>
			<CollapsibleContent>
				Это компонент для создания раскрывающихся секций контента. Пользователи могут нажать на триггер, чтобы
				показать или скрыть содержимое. Полезен для FAQ, меню навигации и других интерактивных элементов.
			</CollapsibleContent>
		</Collapsible>
	);
};
```

### Компоненты

- `Collapsible` - корневой компонент
- `CollapsibleTrigger` - кнопка для открытия/закрытия контента
- `CollapsibleContent` - контейнер с контентом, который показывается/скрывается
