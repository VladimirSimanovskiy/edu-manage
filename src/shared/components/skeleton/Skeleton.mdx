# Skeleton

Компонент Skeleton используется для отображения состояния загрузки контента.

- Компонент основан на [shadcn](https://ui.shadcn.com/docs/components/skeleton)

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=64-243&p=f&t=XBXCrkIqicd91Pbk-0)

### Базовое использование

<Canvas of={SkeletonStories.Default} />

```tsx
import { Skeleton } from "ics-ui-kit/components/skeleton/Skeleton";

export const Basic = () => {
	return <Skeleton className="h-[40px] w-[400px]" />;
};
```
