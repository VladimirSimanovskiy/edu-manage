# Loader

`Loader` представляет собой индикатор загрузки с анимированной иконкой. Компонент обеспечивает единообразное отображение состояния загрузки во всем приложении.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=18589-27777&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Loader } from "ics-ui-kit/components/loader";

export const WithText = () => {
	return <Loader>Loading...</Loader>;
};
```
