# FeatureIcon

`FeatureIcon` представляет собой стилизованную версию иконки с фоном и рамкой. Используется для выделения важных элементов интерфейса и создания визуальных акцентов.

- Использует иконки из библиотеки `Lucide`
- Основные сценарии использования:
    - Выделение ключевых функций
    - Визуальные акценты
    - Декоративные элементы

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=1-433&p=f&m=dev).

### Базовое использование

<Canvas of={ComponentStories.Base} />

```tsx
import { FeatureIcon } from "ics-ui-kit/components/icon";
import { Flower } from "lucide-react";

export const Basic = () => {
	return <FeatureIcon icon={Flower} />;
};
```
