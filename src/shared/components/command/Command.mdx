# Command

### Описание

`Command` - это компонент командной палитры, который предоставляет интерфейс для быстрого поиска и выполнения команд.

### Общая информация

- Компонент использует библиотеку [cmdk](https://cmdk.paco.me/) для создания командных палитр
- Использует основан на [Shadcn UI](https://ui.shadcn.com/docs/components/command)

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=60-436&p=f&t=wLWXj89SOdjBVig7-0)

- Поддержка клавиатуры: полная навигация стрелками, Enter для выбора, Escape для закрытия
- Автоматическая фильтрация элементов при вводе текста
- Поддержка отображения горячих клавиш

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import {
	Command,
	CommandInput,
	CommandList,
	CommandEmpty,
	CommandGroup,
	CommandItem
} from "ics-ui-kit/components/command";
import { Calendar, Mail, FileText } from "lucide-react";
import { Icon } from "ics-ui-kit/components/icon";

export const Basic = () => {
	return (
		<Command className="shadow-md">
			<CommandInput placeholder="Поиск команд..." />
			<CommandList>
				<CommandEmpty>Ничего не найдено.</CommandEmpty>
				<CommandGroup heading="Предложения">
					<CommandItem>
						<Icon icon={Calendar} />
						<span>Календарь</span>
					</CommandItem>
					<CommandItem>
						<Icon icon={Mail} />
						<span>Почта</span>
					</CommandItem>
					<CommandItem>
						<Icon icon={FileText} />
						<span>Документы</span>
					</CommandItem>
				</CommandGroup>
			</CommandList>
		</Command>
	);
};
```
