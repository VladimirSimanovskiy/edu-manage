# Shortcut

`Shortcut` - компонент используется для отображения горячих клавиш.

## Рекомендации

- Используйте обозначения относительно раскладки операционной системы:

	- macOS: `⌘K`, `⌥K`, `⇧K`
	- Windows: `Ctrl K`, `Alt K`, `Shift K`

- Используйте пробел между клавишами для раскладки под Windows. Например: `Ctrl K`.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=16736-6416&m=dev)

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { Shortcut } from "ics-ui-kit/components/shortcut";

export const Basic = () => {
	return <Shortcut>⌘</Shortcut>;
};
```
