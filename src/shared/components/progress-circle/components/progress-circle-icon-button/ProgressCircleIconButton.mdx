# ProgressCircleIconButton

`ProgressCircleIconButton` - это специализированная кнопка-иконка, предназначенная для использования с компонентом `ProgressCircle`.

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=24273-76456&m=dev)

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { ProgressCircleIconButton } from "ics-ui-kit/components/progress-circle";
import { RotateCcw } from "lucide-react";

export const Basic = () => {
	return <ProgressCircleIconButton icon={RotateCcw} onClick={() => console.log("Clicked!")} />;
};
```
