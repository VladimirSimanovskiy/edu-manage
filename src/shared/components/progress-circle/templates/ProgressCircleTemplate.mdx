# ProgressCircleTemplate

`ProgressCircleTemplate` - готовый к использованию шаблон кругового индикатора прогресса с заголовком и описанием.

- Компонент основан на базовых компонентах `ProgressCircle`, `ProgressCircleTitle`, `ProgressCircleDescription` и `ProgressCircleIconButton`

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/01_UI-kit?node-id=24234-211623&m=dev)

### Базовое использование

<Canvas of={ProgressCircleTemplateStories.Default} />

```tsx
import { ProgressCircleTemplate } from "ics-ui-kit/components/progress-circle";

export const Basic = () => {
	return <ProgressCircleTemplate value={50} title="50%" description="Загрузка файла" size="md" />;
};
```
