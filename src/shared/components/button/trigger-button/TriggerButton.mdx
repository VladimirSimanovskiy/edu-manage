# TriggerButton

`TriggerButton` - специализированная кнопка, которая меняет свой внешний вид при открытии/закрытии связанных с ней элементов.

- Компонент основан на `Button` и расширяет его функциональность для работы с состояниями
- Компонент автоматически применяет визуальные стили в зависимости от состояния `data-state`

[Ссылка на Figma](https://www.figma.com/design/lc2aOtrukFLJQwxxgH3Cgm/UI-kit?node-id=24135-105473&m=dev)

### Базовое использование

<Canvas of={ComponentStories.Default} />

```tsx
import { TriggerButton } from "ics-ui-kit/components/button";

export const Basic = () => {
	return <TriggerButton data-state="closed">Открыть меню</TriggerButton>;
};
```
